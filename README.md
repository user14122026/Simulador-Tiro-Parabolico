# Simulador-Tiro-Parabolico

Simulador desarrollado en Python para representar el movimiento de un proyectil.

Permite introducir:

velocidad inicial
ángulo de lanzamiento
altura inicial
planeta (gravedad)

El programa calcula:

trayectoria
altura máxima
alcance horizontal

Además representa la trayectoria mediante una gráfica.

Tecnologías:
Python
NumPy
Matplotlib


Lo que he aprendido:
Aplicar las ecuaciones del movimiento parabólico.
Representar datos mediante gráficas.
Organizar un programa en funciones.
Utilizar listas y cálculos numéricos.
Trabajar con matplotlib.
